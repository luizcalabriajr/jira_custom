# Jira Custom Integration for Odoo 18

Integração direta entre **Odoo 18 Community** e **Jira Cloud**, utilizando a API REST oficial do Jira e Webhooks para sincronização automática de issues.

Este módulo foi criado para substituir o antigo *connector_jira* da OCA, que ainda não possui versão compatível com o Odoo 18.

---

## 🚀 Funcionalidades

- Criar issues no Jira diretamente pelo Odoo  
- Abrir issues do Jira em nova aba  
- Receber atualizações do Jira via Webhook  
- Sincronização automática de status  
- Modelo simples e extensível (`jira.issue`)  
- Totalmente compatível com Odoo 18  
- Sem dependências externas (como queue_job ou connector da OCA)
